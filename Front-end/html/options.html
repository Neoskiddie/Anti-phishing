About

The purpose of the project is to create a browser extension that prevents users from loading phishing websites. The project consists of two software components. The first component is a browser extension the user interacts with. The second component is a server with a machine learning model analysing the phishing cues. When a user makes a request it is forwarded to the server. The server then analysis the features of a given website and pass them to the machine learning model. The model returns information on how likely is the website phishing. If the result from the server has high confidence it is phishing, a user is blocked from loading. Several models were trained with a different set of features to find the most effective model.

Use guide

Currently the extension can only be enabled and disabled. No aditional settings are available.
